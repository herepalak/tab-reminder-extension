🧠 Tab Reminder - Chrome Extension

> Never forget why you opened a tab again!

 ✨ Features
- 📝 Save a reason for every tab you open
- ⏰ Set reminders (5 min to 2 hours)
- 📌 View all saved tabs in one place
- 🔔 Get Chrome notifications when reminder fires

 🚀 Installation
1. Clone this repo
2. Go to `chrome://extensions`
3. Enable **Developer Mode**
4. Click **Load unpacked** → select this folder

 🛠️ Built With
- HTML, CSS, JavaScript
- Chrome Extensions API (Manifest V3)
